
# MH.ModLoader
O projeto está no processo de mapear as funções e valores do jogo, posteriormente irei separar as funções do SDK de forma correta, por enquanto está em um arquivo game.cpp/.h E O PROJETO AINDA NÃO FOI TERMINADO. 

# Exemplo
```cpp
if (Manhunt::PlayerState() == PLAYER_INGAME)
{
    Manhunt::Visual::GameText("Manhunt Mod Loader: initialized!");
    Manhunt::Visual::GameTextLeft(1, "Manhunt Mod Loader: initialized!");
    Manhunt::ToggleFPS(true);
    Manhunt::ToggleMemInfo(true);
}
```
